# GFETM
GFETM: Genome Foundation-based Embedded Topic Model for scATAC-seq Modeling

## Installation
```
git clone https://github.com/fym0503/GFETM.git
cd GFETM
conda create -n GFETM_env python=3.8.10
pip install -r requirements.txt
pip install -e .
```
